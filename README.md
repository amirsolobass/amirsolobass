## 🛠 Development

I build across the stack: CLI tools, REST APIs, AI agents, browser games, and static site generators. My main languages are Python, TypeScript, and JavaScript. I'm drawn to projects where I have to understand how things work from first principles.

Most of my technical foundation comes from [Boot.dev](https://www.boot.dev/), where I worked through backend engineering, systems programming, databases, and more.
